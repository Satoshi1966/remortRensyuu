# docs/ フォルダへの画像差し替え方法

1. お手元の UML / ER 図 / 画面キャプチャを `docs/` にドラッグ＆ドロップ。
2. 既存のプレースホルダー画像（1x1 PNG）を同名で上書きしてください：
   - er_diagram.png
   - uml_sequence.png
   - usecase.png
   - class_diagram.png
   - login_screen.png
   - list_screen.png
3. README.md の該当箇所に自動で反映されます（相対パス参照のため）。

*補足*: 画像ファイル名を変えたい場合は、README.md 内の参照パス（`./docs/xxx.png`）も合わせて修正してください。
